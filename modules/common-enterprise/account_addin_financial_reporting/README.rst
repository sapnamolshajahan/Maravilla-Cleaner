Add-in Financial Reporting
==========================

This application provides flexible and dynamic financial reporting based on your Chart of Accounts.

It can be used to produce "as at" Balance Sheet and P&L financial reports with the following columns:

* **Actual Period** - Report a single period at any point relative to the report "as at" period.
* **Actual YTD** - Report financial YTD for P&L reports.
* **Budget Period** - Budgets for a single period at any point relative to the report "as at" period using the Financial Budgets module.
* **Variance** - Variance between any two report columns
* **Variance Percent** - Variance between any two report columns as a percentage.

Other features
--------------

* **Rounding** - Choose rounding to dollars and cents, or dollars.
* **Account Chart** - Choose any chart of accounts you have set up.
* **Re-use** - Save your report definitions so you can re-use them.
* **Multiple Outputs** - Output to CSV, PDF or Excel.
