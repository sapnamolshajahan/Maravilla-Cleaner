Automatic Packing Slips/Picking List Print
==========================================

Automatically generate and print packing-slips and picking-lists:

   * Picking List when sales orders are confirmed
   * Packing Slip/Putaway when shipments are confirmed

Warehouse
---------
    Inventory > Configuration > Warehouse Management > Warehouses

    Packing Slip/Put-away Printer
        Printer used by the warehouse for packing slips. If this is empty,
        auto packing-slip printing is disabled for this warehouse.

    Picking List Printer
        Printer used by the warehouse for picking lists. If this is empty,
        auto picking-list printing is disabled for this warehouse.
