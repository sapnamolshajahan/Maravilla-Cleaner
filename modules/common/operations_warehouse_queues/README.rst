Warehouse Print Queues
======================

A Technical Module that defines packing-slip and picking-list queues.
This module provides a base for other modules providing print
functionality involving a Warehouse.

Warehouse
---------
    Inventory > Configuration > Warehouse Management > Warehouses

    Packing Slip/Put-away Printer
        Printer used by the warehouse for packing slips.

    Picking List Printer
        Printer used by the warehouse for picking lists.
