Base Generic Changes
====================

A number of general customisations common to OptimySME modules.

Attachment Storage
------------------

Sets the default-storage for attachments to **database**.

Data Access
-----------

Allow Country and Currency data to be accessed from the Settings Menu

Mail Servers
------------

Configure default servers explicitly rather than relying on default-code behaviour

Precisions
----------

Define standard precisions used by OptimySME.

Hidden Menus
------------

Add a technical group to hold hidden menus (rather than deleting them)
